# yozlog Homebrew Tap

My personal collection of Homebrew formulas.

## Installation

```bash
brew tap yozlog/tap
```

## Available Formulas

- [**bw-fzf-copy**](https://github.com/yozlog/bw-fzf-copy/): A script-based TUI tool that uses fzf to quickly search and copy items from Bitwarden CLI.

```bash
brew install bw-fzf-copy
```
