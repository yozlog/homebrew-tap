# yozlog Homebrew Tap

My personal collection of Homebrew formulas.

## Installation

```bash
brew tap yozlog/tap
```

## Tools by me

### Formulas

- [**bw-fzf-copy**](https://github.com/yozlog/bw-fzf-copy/): A script-based TUI tool that uses fzf to quickly search and copy items from Bitwarden CLI.

```bash
brew install bw-fzf-copy
```

- [**tui-chan**](https://github.com/yozlog/tui-chan): Terminal [4chan](https://www.4chan.org/) TUI client. (Forked from [tuqqu/tui-chan](https://github.com/tuqqu/tui-chan))

```bash
brew install tui-chan
```

## Third-party Tools

### Formulas

- [**dragterm**](https://github.com/Wevah/dragterm): Drag files from the command line on macOS.

```bash
brew install dragterm
```

- [**smassh**](https://github.com/kraanzu/smassh): TUI-based typing test application inspired by the well-known typing site [Monkeytype](https://monkeytype.com/).

```bash
brew install smassh
```

### Cask

- [**skills-manager**](https://github.com/xingkongliang/skills-manager): AI Agent Skills Manager.

```bash
brew install --cask skills-manager
```
