# yozlog Homebrew Tap

개인용 Homebrew Formula 모음입니다.

## 설치 방법

```bash
brew tap yozlog/tap
```

## 내가 개발한 도구

### Formulas

- [**bw-fzf-copy**](https://github.com/yozlog/bw-fzf-copy/): fzf를 사용하여 Bitwarden 보관함 항목을 빠르게 검색하고 복사할 수 있는 스크립트 기반 TUI 도구입니다.

```bash
brew install bw-fzf-copy
```

- [**tui-chan**](https://github.com/yozlog/tui-chan): 터미널 [4chan](https://www.4chan.org/) TUI 클라이언트. ([tuqqu/tui-chan](https://github.com/tuqqu/tui-chan) 포크)

```bash
brew install tui-chan
```

## 서드파티 도구

### Formulas

- [**dragterm**](https://github.com/Wevah/dragterm): macOS 명령줄에서 파일을 드래그합니다.

```bash
brew install dragterm
```

### Cask

- [**skills-manager**](https://github.com/xingkongliang/skills-manager): AI Agent 스킬 관리 도구입니다.

```bash
brew install --cask skills-manager
```
